# KidLang Examples

Welcome to the KidLang examples directory! Here you'll find various programs to help you learn KidLang programming.

## How to Run

```bash
kidlang examples/01_hello_world.kid
```

## Example Programs

### Beginner Level

1. **01_hello_world.kid** - Your first program! Prints greetings to the screen.
2. **02_simple_math.kid** - Learn basic math operations with boxes.
3. **03_user_input.kid** - Get input from users and use it in your program.

### Intermediate Level

4. **04_guessing_game.kid** - Interactive number guessing game with random numbers.
5. **05_calculator.kid** - Full-featured calculator with multiple operations.
6. **06_countdown.kid** - Countdown timer using loops and sleep.
7. **07_multiplication_table.kid** - Generate multiplication tables.
8. **08_temperature_converter.kid** - Convert between Celsius and Fahrenheit.

### Advanced Level

9. **09_math_quiz.kid** - Test your math skills with a 10-question quiz.
10. **10_todo_list.kid** - Manage tasks with file operations.
11. **11_random_story.kid** - Generate funny random stories.
12. **12_simple_game.kid** - Adventure game with health, gold, and random events.

## Learning Path

**Week 1:**
- Start with examples 1-3
- Understand boxes, printing, and user input

**Week 2:**
- Try examples 4-6
- Learn loops, conditions, and random numbers

**Week 3:**
- Work on examples 7-9
- Practice more complex logic and calculations

**Week 4:**
- Challenge yourself with examples 10-12
- Build complete programs with multiple features

**Turkish Speakers:**
- Examples 13-14 show how to use KidLang in Turkish!

## Tips for Learning

1. **Read the code** - Each example has comments explaining what it does
2. **Run the program** - See how it works
3. **Modify it** - Change numbers, messages, or logic
4. **Break it** - Try to make it fail, then fix it
5. **Create your own** - Use examples as templates for your ideas

## Challenge Yourself

After completing these examples, try to:
- Combine features from different programs
- Add new features to existing programs
- Create your own unique programs
- Help others learn by creating more examples!

## Need Help?

- Read the [Programming Guide](../PROGRAMMING_GUIDE.md)
- Check error messages carefully
- Use `--debug` flag to see what's happening: `kidlang --debug myfile.kid`
- Start simple and add complexity gradually
